# Obtain a feature matrix (as seurat object) given an seurat object having Fragments associated

Obtain a feature matrix (as seurat object) given an seurat object having
Fragments associated

## Usage

``` r
CountGrangeFeatures(adata, grangeGene)
```

## Arguments

- adata:

  Seurat object, with FeatureMatrix

- grangeGene:

  A grange object telling where to count

## Value

A Seurat AssayObject
