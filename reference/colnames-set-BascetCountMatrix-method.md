# Set column names of a BascetCountMatrix (feature names)

Set column names of a BascetCountMatrix (feature names)

## Usage

``` r
# S4 method for class 'BascetCountMatrix'
colnames(x) <- value
```

## Arguments

- x:

  A BascetCountMatrix object

- value:

  Character vector of new column names

## Value

The modified BascetCountMatrix
