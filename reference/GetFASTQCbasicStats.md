# From aggregated FASTQC data, get basic statistics for overlay on UMAP etc

From aggregated FASTQC data, get basic statistics for overlay on UMAP
etc

## Usage

``` r
GetFASTQCbasicStats(aggrFastqData, readnum = c(1, 2))
```

## Arguments

- aggrFastqData:

  Aggregated FASTQ data

- readnum:

  1 or 2, for R1 or R2

## Value

A data.frame
