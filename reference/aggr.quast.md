# Callback function for aggregating QUAST data. To be called from BascetAggregateMap

Callback function for aggregating QUAST data. To be called from
BascetAggregateMap

## Usage

``` r
aggr.quast(bascetFile, cellID, bascetInstance)
```

## Arguments

- bascetFile:

  An opened Bascet file

- cellID:

  Cell ID

- bascetInstance:

  A Bascet instance

## Value

Data for one cell
