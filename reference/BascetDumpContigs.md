# Store all contigs in an output directory, as cell_id.fa

Store all contigs in an output directory, as cell_id.fa

## Usage

``` r
BascetDumpContigs(
  bascetRoot,
  inputName = "skesa",
  listCells,
  outputDir,
  bascetInstance
)
```

## Arguments

- bascetRoot:

  The root folder where all Bascets are stored

- inputName:

  Name of input shard (should hold contigs)

- listCells:

  List of cells to extract

- outputDir:

  Directory to store FASTA in

- bascetInstance:

  A Bascet instance

## Value

Nothing
