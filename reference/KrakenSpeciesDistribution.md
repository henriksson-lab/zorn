# Using a KRAKEN2 count matrix, produce a "kneeplot" of species

Using a KRAKEN2 count matrix, produce a "kneeplot" of species

## Usage

``` r
KrakenSpeciesDistribution(adata, useAssay = "kraken")
```

## Arguments

- adata:

  A Seurat object

- useAssay:

  Which assay to use

## Value

A ggplot object
