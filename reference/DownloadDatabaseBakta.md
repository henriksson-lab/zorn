# Download a database for Bakta

Download a database for Bakta

## Usage

``` r
DownloadDatabaseBakta(
  dbdir,
  dbtype = c("light", "full"),
  bascetInstance = GetDefaultBascetInstance()
)
```

## Arguments

- dbdir:

  Directory path to download database to (string)

- dbtype:

  Database type: "light" or "full"

- bascetInstance:

  A Bascet instance
