# Bascet instance settings

Runtime settings for the underlying Bascet installation.

## Slots

- `bin`:

  Name or path of the Bascet binary.

- `tempdir`:

  Directory for temporary files.

- `prependCmd`:

  Command prefix, for example for container execution.

- `containerMem`:

  Memory reserved for container overhead.

- `logLevel`:

  Log level passed to Bascet.

- `logToFile`:

  Whether Bascet logs are written to files.
