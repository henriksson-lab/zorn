# Set row names of a BascetCountMatrix (cell names)

Set row names of a BascetCountMatrix (cell names)

## Usage

``` r
# S4 method for class 'BascetCountMatrix'
rownames(x) <- value
```

## Arguments

- x:

  A BascetCountMatrix object

- value:

  Character vector of new row names

## Value

The modified BascetCountMatrix
