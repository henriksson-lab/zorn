# Read histogram of KMERs, the output of BascetMakeMinhashHistogram

Read histogram of KMERs, the output of BascetMakeMinhashHistogram

## Usage

``` r
BascetReadMinhashHistogram(bascetRoot, inputName = "minhash_hist.csv")
```

## Arguments

- bascetRoot:

  The root folder where all Bascets are stored

- inputName:

  Name of input shard

## Value

KMER histogram
