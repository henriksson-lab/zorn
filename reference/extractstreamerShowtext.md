# extract streamer: get content of file, assumed to be text (or this function crashes)

extract streamer: get content of file, assumed to be text (or this
function crashes)

## Usage

``` r
extractstreamerShowtext(p, fname, verbose = FALSE)
```

## Arguments

- p:

  Streamer process

- fname:

  Name of file

- verbose:

  Print additional information, primarily to help troubleshooting

## Value

The text, divided by line
