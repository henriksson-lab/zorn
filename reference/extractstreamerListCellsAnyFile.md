# extract streamer: list all cells in a given file

extract streamer: list all cells in a given file

## Usage

``` r
extractstreamerListCellsAnyFile(p, fname, verbose = FALSE)
```

## Arguments

- p:

  Streamer process

- fname:

  Name of file

- verbose:

  Print additional information, primarily to help troubleshooting

## Value

List of cells
