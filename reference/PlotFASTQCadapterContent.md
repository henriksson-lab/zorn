# From aggregated FASTQC data, plot adapter content

From aggregated FASTQC data, plot adapter content

## Usage

``` r
PlotFASTQCadapterContent(aggrFastqData, readnum = c(1, 2))
```

## Arguments

- aggrFastqData:

  Aggregated FASTQ data

- readnum:

  1 or 2, for R1 or R2

## Value

A ggplot object
