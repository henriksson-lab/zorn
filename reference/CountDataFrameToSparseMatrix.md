# Count entries in long format data frame and return as a sparse matrix

Count entries in long format data frame and return as a sparse matrix

## Usage

``` r
CountDataFrameToSparseMatrix(dat, rowname, colname)
```

## Arguments

- dat:

  A data.frame

- rowname:

  Column to use as matrix row name

- colname:

  Column to use as matrix column name

## Value

A dgCMatrix sparse matrix
