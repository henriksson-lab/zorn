# Get list of cells in a Bascet

Get list of cells in a Bascet

## Usage

``` r
BascetCellNames(bascetRoot, bascetName, bascetInstance)
```

## Arguments

- bascetRoot:

  The root folder where all Bascets are stored

- bascetName:

  Name of the bascet

- bascetInstance:

  A Bascet instance

## Value

Vector of cell names as strings
