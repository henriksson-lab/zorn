# Get list of cells in a Bascet – streamer provided

Get list of cells in a Bascet – streamer provided

## Usage

``` r
BascetCellNames_withstreamer(bascetRoot, bascetName, streamer)
```

## Arguments

- bascetRoot:

  The root folder where all Bascets are stored

- bascetName:

  Name of the bascet

- streamer:

  Bascet streamer instance

## Value

Vector of cell names as strings
