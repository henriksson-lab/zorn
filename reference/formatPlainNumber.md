# Parse a string with a size, such as 1g, 1m, 1k, or just 123 (bytes)

Parse a string with a size, such as 1g, 1m, 1k, or just 123 (bytes)

## Usage

``` r
formatPlainNumber(s)
```

## Arguments

- s:

  Size as string. If numeric, it is just returned

## Value

Size in mb, as integer in a string (long format)
