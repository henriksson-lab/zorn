# Helper function to take an array of elements and split it randomly into a number of subset lists

Helper function to take an array of elements and split it randomly into
a number of subset lists

## Usage

``` r
shellscriptSplitArrayIntoListRandomly(arr, num_divide)
```

## Arguments

- arr:

  Array of elements to subdivide

- num_divide:

  Number of subdivions

## Value

List of arrays
