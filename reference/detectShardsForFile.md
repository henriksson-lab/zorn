# Helper function: Figure out which shards belong together given root input name and extension i.e. root/name.##.ext

Helper function: Figure out which shards belong together given root
input name and extension i.e. root/name.##.ext

## Usage

``` r
detectShardsForFile(bascetRoot, inputName)
```

## Arguments

- bascetRoot:

  The root folder where all Bascets are stored

- inputName:

  Name of input shard

## Value

List of files that correspond to the shard
