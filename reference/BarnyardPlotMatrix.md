# Produce a matrix of Barnyard plots, i.e., counts for one species vs another, for all combinations of species.

Produce a matrix of Barnyard plots, i.e., counts for one species vs
another, for all combinations of species.

## Usage

``` r
BarnyardPlotMatrix(adata)
```

## Arguments

- adata:

  A Seurat object with the DefaultAssay holding counts per species (or
  similar)

## Value

a ggarranged set of ggplots
