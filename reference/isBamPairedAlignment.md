# Figure out if a BAM-file is a paired alignment or not

Figure out if a BAM-file is a paired alignment or not

## Usage

``` r
isBamPairedAlignment(fname, bascetInstance = GetDefaultBascetInstance())
```

## Arguments

- fname:

  Name of BAM-file

- bascetInstance:

  A Bascet instance

## Value

TRUE if the BAM-file is a paired alignment
