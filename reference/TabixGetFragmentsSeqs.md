# Using Tabix, get list of sequences in a fragment file

Using Tabix, get list of sequences in a fragment file

## Usage

``` r
TabixGetFragmentsSeqs(fragpath, bascetInstance = GetDefaultBascetInstance())
```

## Arguments

- fragpath:

  List to fragment file

- bascetInstance:

  A Bascet instance

## Value

The raw result of tabix
