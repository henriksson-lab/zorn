# From aggregated FASTQC data, get overall pass-fail statistics for overlay on UMAP etc

From aggregated FASTQC data, get overall pass-fail statistics for
overlay on UMAP etc

## Usage

``` r
GetFASTQCpassfailStats(aggrFastqData, readnum = c(1, 2))
```

## Arguments

- readnum:

  1 or 2, for R1 or R2

## Value

A data.frame
