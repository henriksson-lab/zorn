# From a Signac chromatin assay with fragments, for each cell, count how many reads per chromosome

From a Signac chromatin assay with fragments, for each cell, count how
many reads per chromosome

## Usage

``` r
FragmentCountsPerChrom(chromAssay, bascetInstance = GetDefaultBascetInstance())
```

## Arguments

- chromAssay:

  A ChromatinAssay

- bascetInstance:

  A Bascet instance

## Value

a FeatureMatrix
