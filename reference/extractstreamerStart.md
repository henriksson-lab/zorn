# extract streamer: create an instance

extract streamer: create an instance

## Usage

``` r
extractstreamerStart(
  fname = NULL,
  verbose = FALSE,
  bascetInstance = GetDefaultBascetInstance()
)
```

## Arguments

- fname:

  File to stream (can be NULL if set later)

- verbose:

  Print additional information, primarily to help troubleshooting

- bascetInstance:

  A Bascet instance

## Value

The process
