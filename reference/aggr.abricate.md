# Callback function for aggregating ABRicate data for each cell. To be called from BascetAggregateMap

Callback function for aggregating ABRicate data for each cell. To be
called from BascetAggregateMap

## Usage

``` r
aggr.abricate(bascetFile, cellID, bascetInstance)
```

## Arguments

- bascetFile:

  An opened Bascet file

- cellID:

  Cell ID

- bascetInstance:

  A Bascet instance

## Value

Data for one cell
