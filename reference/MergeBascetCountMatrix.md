# Merge a list of count matrices as produced by Bascet

Merge a list of count matrices as produced by Bascet

## Usage

``` r
MergeBascetCountMatrix(listInput, prependName = FALSE, verbose = FALSE)
```

## Arguments

- listInput:

  List of count matrices

- prependName:

  If TRUE, prepend name of data in list to each cell

- verbose:

  Print additional information, primarily to help troubleshooting

## Value

BascetCountMatrix
